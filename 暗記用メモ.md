# SQL分類一覧

* データ検索
  * SELECT：参照、検索
* データ操作言語/DML/Data Manipulation Language
  * INSERT：追加、挿入
  * UPDATE：更新
  * DELETE：削除
* データ定義言語/DDL/Data Definition Language
  * CREATE：オブジェクト作成
  * DROP：オブジェクト削除
  * ALTER：オブジェクト定義変更
  * TRUNCATE：テーブル内の全データ削除、切り捨て
* トランザクション制御
  * COMMIT：トランザクション確定
  * ROLLBACK：トランザクション取り消し
* データ制御言語/DCL/Data Control Language
  * GRANT：権限付与
  * REVOKE：権限取り消し、剥奪
